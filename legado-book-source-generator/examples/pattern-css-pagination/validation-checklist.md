# Legado 验收清单

1. 导入 `book-source.json`
2. 搜索 `凡人修仙`
3. 验证搜索结果中的书名、作者、封面、详情链接均正常显示
4. 进入任一书籍详情页，验证书名、作者、简介、封面正常展示
5. 打开目录，验证目录首页可加载
6. 翻到目录下一页，验证 `nextTocUrl` 生效
7. 打开任一存在分页的章节，验证正文第一页正常显示
8. 继续翻页，验证同章分页会自动拼接或继续加载，不跳到下一章
9. 再打开至少一章正文，验证规则对第二个样本仍然成立
10. 若失败，优先回溯失败链路:
    - 搜索失败: 搜索列表选择器、详情链接补全
    - 目录失败: 目录分页选择器
    - 正文失败: 正文主体容器与 `nextContentUrl` 的同章判定
